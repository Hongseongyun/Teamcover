## 02. 화면설계서

Teamcover 서비스의 **화면 설계서** 템플릿입니다.  
각 화면에 대한 구조, 컴포넌트, 상태, 예외 케이스 등을 정의합니다.

### 2. 화면 설계 상세 템플릿 (예시)

#### P-002. 회원 목록 화면 (`/members`)

- **목적**

  - 클럽/리그의 전체 회원을 조회하고, 검색/필터/정렬/엑셀 다운로드 등을 수행한다.

- **주요 기능**

  - 회원 리스트 조회 (페이지네이션)
  - 검색(이름, 연락처, 회원번호 등)
  - 필터(활동/비활동, 티어, 가입일 등)
  - 엑셀 다운로드
  - 회원 상세 이동

- **레이아웃 구조**

  - 상단: 페이지 제목, 전체 회원 수, 주요 필터 요약
  - 본문: 테이블(회원 리스트)
  - 하단: 페이지네이션

- **컴포넌트 구성 (React 기준 예시)**

  - `MembersPage`
    - `MembersFilterBar`
    - `MembersTable`
    - `Pagination`

- **상태/데이터**

  - `filters`: { keyword, status, tier, ... }
  - `members`: Member[] (API 응답)
  - `page`, `pageSize`, `totalCount`

- **API 연동**

  - `GET /api/members`
    - query: `page`, `pageSize`, `keyword`, `status`, `tier` 등
    - response: `items`, `totalCount`

- **예외/에러 케이스**

  - 검색 결과 없음
  - API 오류 (네트워크 오류, 서버 오류)
  - 권한 부족(로그인 만료 등)

- **화면 시안(Figma 등) 링크**
  - `[링크를 여기에 삽입]`
