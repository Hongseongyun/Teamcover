## 04. 서비스 정책서

Teamcover 서비스의 **운영/서비스 정책**을 정리하는 문서입니다.  
회원, 요금제, 데이터 보존, 알림 등 운영 규칙을 명문화합니다.

### 1. 서비스 개요

- **서비스 대상**
  - 볼링 클럽/리그 운영자, 스태프, 일반 회원
- **주요 기능**
  - 회원 관리, 출석/성적 관리, 회비/정산, 알림/공지, 통계 등

### 2. 회원/계정 정책

- **회원 유형**
  - 클럽 관리자, 운영진(스태프), 일반 회원
- **가입/탈퇴**
  - 가입 방법: 초대 링크 / 공개 신청 / 관리자 등록 등
  - 탈퇴 시 처리: 데이터 보존 기간, 정산 이력 보관 여부
- **권한 관리**
  - 어떤 역할이 어떤 메뉴/기능에 접근 가능한지 정의 (IA와 연계)

### 3. 요금제/결제 정책 (필요 시)

- **요금제 구조**
  - 무료/유료 플랜 구분
  - 동시 사용 인원, 클럽 수, 기능 제한 등
- **결제/환불**
  - 청구 주기(월/년), 결제 수단
  - 환불 정책(부분 환불/전액 환불 불가 등)

### 4. 데이터/보안 정책

- **데이터 보존 기간**
  - 탈퇴 회원 데이터 보관 기간
  - 로그/이력 데이터 보관 기간
- **개인정보 처리**
  - 이름, 연락처, 점수/성적 데이터 등
  - 제3자 제공 여부(없음/있음 + 목적)
- **접근 제어**
  - 관리자/운영진의 데이터 조회 범위

### 5. 알림/메시지 정책

- **알림 채널**
  - 이메일, 웹 푸시, In-App 알림 등
- **발송 규칙**
  - 어떤 이벤트에 대해 어떤 알림을 발송하는지 정의
  - 야간 발송 제한, 발송 빈도 제한 등

### 6. 운영/지원 정책

- **문의 채널**
  - 이메일, 카카오톡 채널, 기타
- **운영 시간**
  - 평일/주말, 응답 SLA
- **장애 공지**
  - 장애 발생 시 공지 채널/절차

### 7. 정책 변경 관리

- 정책 변경 시 버전과 날짜를 기록하고, 주요 변경 사항을 요약합니다.
